# CTS Digital Nurture 5.0 – DotNet FSE Learning Repository

## Overview

This repository contains my learning journey, practice exercises, hands-on implementations, assessments, and project artifacts completed as part of the Cognizant Digital Nurture 5.0 (DN 5.0) DotNet Full Stack Engineer Program.

The repository is organized into two major learning tracks:

### 1. Upskilling

The Upskilling program focuses on strengthening foundational programming and web development skills.

**Module 1**
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery

**Module 2**
- ANSI SQL using MySQL

**Module 3**
- C#
- ADO.NET

### 2. Deep Skilling

The Deep Skilling program focuses on advanced Full Stack Engineering concepts and enterprise application development.

**Modules Covered**
- Design Principles and Patterns
- Data Structures and Algorithms
- SQL for Application Development
- Unit Testing
- Entity Framework Core
- ASP.NET Core Web API
- Microservices
- Angular
- Application Debugging
- Git
- DevOps and CI/CD
- Docker
- Agile Methodology
- GenAI Fundamentals
